Bookbuilder
===

[![npm version](https://badge.fury.io/js/bookbuilder.svg)](https://badge.fury.io/js/bookbuilder)

Bookbuilder for larbreapages.fr with react.js

Install
---

`make install`

Build
---

`make build`

Run
---

`MAIL_PASSWORD=MYPASS STRIPE_SECRET_KEY=MYSECRET make run`
